Common Tags

In HTML there are a few tags that are essential to know. With all these you are required to add "<(tag)>". To close any tag add a "</(tag)>".

The most essential tag is the "html" tag, which you use to open all html files. You put the opening one at the start and the closing one at the end.

"head" is also an important tag as you put your title of your webpage and metadata (a set of data that describes and gives information about other data.) in it.

"body" is where you put headers, text, images, videos etc.

"h" is short for header, it is basically a title, subheader, whatever you wish to call it. You put h and then a number between 1 and 6. The number represents the type of header, is the largest and boldest, 6 is the smallest and weakest.

Images

To inbed a photo in your web page just use the code "<img src="learning_html.jpg" alt="Learning HTML">". Where it says src="learning_html.jpg" that is where you put the name of the image or the URL of the image. alt="Learning HTML" is what you wish to name the image.

If you wish to control the height and width of the image to adjust the size, use the code "<img src="html.png" alt="HTML 5 Logo" style="height:640px; width:800px;">". You can make the dimensions whatever you wish.

Tables

To create a table you need 4 main tags. "<table>" "<tr>" "<th>" "<td>"

"<table>" is what you use to start off the table. It tells the browser that the code following is to be put in a table.

"<tr>" is the row, use this to create a row

"<th>" is the table column header

"<td>" is what is in the cell, the data held inside the cell. Note that they can contain all sorts of HTML elements such as text, images, lists, other tables, etc.



Validation

Validation is where a program runs through your code making sure it is all perfect, it detects any issues and tells you where it is and what is wrong with it. Although most of the errors wont actually affect the outcome of the code it just makes it all proper and correct and perfect

To validate a website to XHTML you must put this at the very top of your code

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
